Банкомат на JavaFX

Учебный проект банкомата с графическим интерфейсом на JavaFX.

Функционал

Вставка карты по номеру
Ввод PIN-кода (3 попытки)
Снятие наличных
Перевод со счёта на карту
Просмотр баланса
История операций (для кредитной карты)
Отмена операции (кнопка "0" или "cancel")
Отчёт о выполненных операциях

Структура

karta – абстрактная карта
debetkarta – дебетовая карта
kreditkarta – кредитная карта
schet – банковский счёт
baza – хранилище данных
upravlenie – контроллер JavaFX
Main – точка входа
